### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sundeepreddy0404/ai-code-agent.git
   cd ai-code-agent
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   Navigate to http://localhost:5173
   ```

## ğŸ¯ Usage

### Writing Code
1. Click "Write Code" or type your request
2. Example: *"Write a function to reverse a string"*
3. AI generates the code automatically
4. Code appears in the editor

### Debugging
1. Write or paste buggy code in the editor
2. Click "Debug" or type *"Debug this code"*
3. AI identifies issues and provides fixed code
4. Review changes and test

### Explaining Code
1. Paste complex code in the editor
2. Click "Explain" or ask *"Explain what this does"*
3. Get line-by-line explanations
4. Understand the logic

### Optimizing
1. Add working code to the editor
2. Click "Optimize" or request *"Optimize this code"*
3. AI suggests performance improvements
4. Compare before and after
